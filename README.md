# Argon

An open source ticket booking app, inspired by [Hi.Events](https://hi.events/).

## Tech Stack

- [Laravel](https://laravel.com/)
- [Vue 3](https://vuejs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Inertia.js](https://inertiajs.com/)

## Features

- Event Management
- Ticket Management
- User Management
- Organizer Management
- Reservation System
- Seat Maps

## Installation

```bash
composer update
npm install
npm run build
php artisan setup
```
